# Клонирование голоса — Инструкция для начинающих

## Что тебе понадобится
- Браузер (Chrome или Firefox)
- Google аккаунт (бесплатный)
- Аудиозапись своего голоса (3-30 секунд)

---

## Вариант 1: Google Colab — РЕКОМЕНДУЮ (бесплатно, без установки)

### Шаг 1: Открой ноутбук
1. Перейди на https://colab.research.google.com
2. Нажми **Файл → Загрузить ноутбук**
3. Загрузи файл `openvoice_v2_russian.ipynb` из этой папки

### Шаг 2: Включи GPU (ВАЖНО!)
1. В меню сверху: **Среда выполнения**
2. → **Сменить среду выполнения**
3. → **Аппаратный ускоритель: T4 GPU**
4. → **Сохранить**

### Шаг 3: Запусти по порядку
- Нажимай Shift+Enter в каждой ячейке сверху вниз
- Или нажми **Среда выполнения → Выполнить всё**
- Каждая ячейка должна показать зелёную галочку ✅

### Шаг 4: Запись эталонного голоса
Запиши аудио где говоришь ЭТОТ текст (в нём много букв Р):
> "Рыжая лиса прыгнула через реку. Красные розы растут в ряду.
> Прораб проверил работу рабочих. Три пирога с творогом."

**Важно:**
- Тихая комната, без эха
- Микрофон телефона подойдёт
- WAV или MP3 формат
- 10-15 секунд оптимально

### Шаг 5: Загрузи файл и введи текст
- В ячейке "Шаг 4" нажми появившуюся кнопку и выбери свой файл
- В ячейке "Шаг 5" замени текст на свой

### Шаг 6: Скачай результат
- Последняя ячейка скачает WAV файл на твой компьютер

## Вариант 2: Онлайн демо (ещё проще, но меньше контроля)

### OpenVoice — официальное демо:
https://huggingface.co/spaces/myshell-ai/OpenVoiceV2

1. Открой ссылку
2. Загрузи аудио своего голоса (Reference Audio)
3. Введи текст
4. Нажми Generate
5. Скачай результат

### Другие онлайн демо:
- XTTS-v2: https://huggingface.co/spaces/coqui/xtts
- F5-TTS: https://huggingface.co/spaces/mrfakename/E2-F5-TTS

## Вариант 3: ElevenLabs — самый простой (но платный после лимита)

1. Зайди на https://elevenlabs.io
2. Зарегистрируйся (бесплатный план: 10 мин/месяц)
3. Нажми **Voice Lab → Add Voice → Instant Voice Clone**
4. Загрузи аудио (минимум 1 минута для хорошего качества)
5. Введи текст → Generate

**Минус:** бесплатный план ограничен, картавость может сгладиться

---

## Советы по записи голоса с картавостью

Для лучшего результата в записи должно быть много слов с [Р]:
- "рыжая рыба ревёт рядом"
- "красные розы растут в ряду"
- "прораб проверил работу"
- "три пирога с творогом"
- "в роще прохладно и сыро"

Чем чётче и отчётливее картавость в записи — тем лучше клон её сохранит.

## Что делать если не работает

| Проблема | Решение |
|---------|---------|
| "GPU не найден" | Среда выполнения → Сменить → T4 GPU |
| Установка зависает | Подожди 5 минут, HuggingFace иногда медленный |
| Ошибка при загрузке модели | Попробуй снова (сетевая ошибка) |
| Голос звучит не похоже | Используй более длинную/чистую запись (15-30 сек) |
| Потеряла картавость | Убедись, что в эталонной записи есть слова с Р |
