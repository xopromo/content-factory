## 📘 ВКонтакте (VK)

### 🎯 Таргетирование по интересам
```
ВК позволяет таргетировать по 10+ категориям интересов.

💡 Лайфхак: Комбинируй broad audience с узким целевым сегментом
   для поиска новых ниш.

📈 Результат: +35% ROI при правильной комбинации аудиторий
```

**Как действовать:**
1. Создай несколько аудиторий
2. Тестируй комбинации интересов
3. Масштабируй лучшие комбинации

### 💰 Оптимизация бюджета в ВК
```
Лучше начинать с малого бюджета (100-500р/день) и масштабировать.

ВК автоматически оптимизирует под конверсии если выставить
нужный event в pixel.

✅ Совет: Используй А/В тесты на 10 разных креативов одновременно
```

**Действия:**
- Начни с $3-7 в день
- Добавь pixel event для конверсии
- Запусти 10 вариантов объявления

### ⏰ Оптимальное время для запуска
```
📊 Пик активности ВК:
   - 19:00-23:00 (вечер) 🌙
   - 12:00-14:00 (обед) 🍽️

📊 Для B2B:
   - 08:00-10:00 (утро) ☀️
   - 14:00-16:00 (полдень)

💡 Совет: Запускай кампании во вторник-четверг, избегай выходных
```

## 🧵 Threads

### 🚀 Вирусный потенциал
```
Threads копирует механику Twitter но с более лояльной аудиторией.

💡 Лайфхак: Короткие, дерзкие посты получают больше reach
   чем длинные.

🎯 Стратегия:
   - Публикуй 3-5 постов в день
   - Провоцируй дискуссии в комментариях
   - Отвечай на каждый комментарий в первый час
```

### #️⃣ Хэштеги и discoverability
```
💡 Оптимальная формула: 5-7 релевантных хэштегов на пост

📊 Структура:
   - 2-3 популярных (#ai, #marketing)
   - 3-4 нишевых (#threadsmktg, #trafficgrowth)

📈 Результат: +250% reach при правильной расстановке хэштегов
```

**Чек-лист хэштегов:**
- [ ] 1-2 очень популярных (>100k постов)
- [ ] 2-3 среднепопулярных (10k-100k)
- [ ] 2-3 нишевых (<10k)

## 🔍 Яндекс Директ

### 🔎 Keyword Research (Wordstat)
```
Яндекс Директ требует DEEP keyword research - используй Wordstat!

💡 Лайфхак: Ищи длинные low-competition фразы (4+ слова)

🎯 Стратегия keyword группировки:
   - 10-15 групп по смыслу
   - 1 уникальное объявление на группу
   - 3-5 ключей на группу
```

**Найти хорошие ключи:**
1. Открой Wordstat
2. Введи broad ключ (например, "рекламное агентство")
3. Сортируй по колво поисков
4. Ищи фразы 4+ слова с 100-500 поисков/месяц
5. Проверь конкуренцию (CPC)

### 💰 Bidding Strategy в Директе
```
Стратегия управления ставками:

1️⃣ Начинай с ручного управления ставками
2️⃣ Затем переходи на автоставки

💡 Лайфхак: Установи max CPC на 30-40% выше средней стоимости клика

📊 Совет: Используй средневзвешенные позиции (4-7 место)
   для лучшего ROI
```

**Расчет стартовой ставки:**
- Средняя стоимость клика (CPC) = X
- Моя ставка = X × 1.35 (35% выше)

### 🚫 Минус-слова и фильтрация
```
Минус-слова - это 50% успеха в Директе!

💡 Лайфхак: Веди таблицу минус-слов на уровне аккаунта

Обязательные исключения:
   - бесплатно
   - курсы
   - урок
   - demo
   - trial
   [добавь свои отрасль-специфичные]

Примеры для конкретных ниш:
   🏠 Недвижимость: -квартира, -дом, -аренда
   📚 Образование: -школа, -университет
   💼 B2B услуги: -лучший, -дешевле
```

### 🎁 Расширения объявлений
```
Используй ВСЕ доступные расширения:
   ✅ Уточнения (до +20% CTR)
   ✅ Быстрые ссылки (до +15% CTR)
   ✅ Цены (до +25% CTR)
   ✅ Телефон (для B2B)

💡 Лайфхак: Расширения могут увеличить CTR на +30%!

📋 Чек-лист:
   - [ ] Добавил 4+ уточнения

   - [ ] Показываю цены (если есть)
   - [ ] Добавил телефон
   - [ ] А/В тестирую варианты
```

## 📊 Универсальные советы

### Во всех платформах

**A/B Тестирование:**
```
- Тестируй минимум 3 варианта creatives
- На каждый - минимум $10-20 бюджета
- Даём минимум 100-200 impressions на вариант
- Побеждает вариант с лучшим CTR и CPA
```

**Оптимизация бюджета:**
```
- Начинай маленько ($5-10)
- Если работает - масштабируй на +20-30% в день
- Следи за всеми метриками (CTR, CPC, ROAS)
- Убивай lo-performing варианты
```

**Метрики для отслеживания:**
```
✅ CTR (Click Through Rate)     - цель: >1.5%
✅ CPC (Cost Per Click)         - оптимизируем вниз
✅ CPM (Cost Per 1000 Imp)      - ниже = лучше
✅ CPA (Cost Per Action)        - чем ниже, тем лучше
✅ ROAS (Return On Ad Spend)    - цель: >3x
```
