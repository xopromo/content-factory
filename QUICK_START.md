# 🚀 БЫСТРЫЙ СТАРТ: Агенты за 30 секунд

## ⚡ САМЫЙ ПРОСТОЙ СПОСОБ

```bash
./new-agent
```

Ответь на 3 вопроса:
1. Как назвать?
2. Для чего он?
3. Насколько сложно?

✅ ГОТОВО! Агент создан.

---

## 📋 ДРУГИЕ КОМАНДЫ

### Создать агента с параметрами

```bash
# Простая версия
bash create-agent.sh "my-agent" "Описание"

# С указанием сложности
bash create-agent.sh "my-agent" "Описание" high
```

### Запустить pipeline (исследование → идеи → критика → вердикт)

```bash
bash run-agent-pipeline.sh "твой вопрос" budget
```

### Запустить одного агента

```bash
@"researcher" исследуй [тема]
@"brainstormer" напиши идеи на основе RESEARCH.md
@"critic" оцени идеи
@"judge" выноси финальный вердикт
```
