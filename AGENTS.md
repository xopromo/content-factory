# 🤖 Агенты с автоматической эволюцией

## Быстрый старт

### Запустить фрилансера (эволюция)
```bash
./evolve-freelancer.sh
```

## Что происходит внутри?

### 1️⃣ Первый запуск
```bash
./evolve-freelancer.sh
```
- ✅ Проверяет: есть ли `venv/`?
- ✅ Если нет → создает его
- ✅ Установит все пакеты если нужно
- ✅ Запускает фрилансера

### 2️⃣ Второй запуск (через неделю)
```bash
./evolve-freelancer.sh
```
- ✅ Находит `venv/` (он остался на диске!)
- ✅ Активирует его
- ✅ Запускает фрилансера
- ✅ Начинается новая эволюция

### 3️⃣ Эволюция между запусками
Каждый раз фрилансер:
- 🧬 Мутирует промпт
- 📊 Тестирует на 5 задачах
- ⭐ Выбирает лучший вариант
- 🏷️ Git-тагирует: `evo-1`, `evo-2`, ...

## Добавить нового агента

Создай свой скрипт:
```bash
# Новый скрипт
cat > my_agent_evolution.py << 'EOF'
from freelancer_a_evolve import MyAgent, MyBenchmark
# ... твой код ...
EOF

# Создай быстрый запуск
echo '#!/bin/bash' > evolve-myagent.sh
echo './run-agent.sh my_agent_evolution.py "$@"' >> evolve-myagent.sh
chmod +x evolve-myagent.sh

# Запусти
./evolve-myagent.sh
```

## FAQ

**Q: Куда установятся пакеты?**
A: В папку `venv/` в твоем проекте

**Q: Если я удалю `venv/`?**
A: Скрипт создаст заново, переустановит пакеты

**Q: Можно ли использовать эти скрипты в других проектах?**
A: Да! Скопируй `run-agent.sh` и адаптируй под свои нужды

**Q: Если интернета нет, будет ли работать?**
A: Да! Пакеты уже установлены в `venv/`. Работает офлайн.
