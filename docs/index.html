Агент запускается через GitHub Actions каждый день в 09:00 UTC. Выполняет 14 поисковых запросов по VK, Threads и Яндекс.Директ, сохраняет результаты в docs/research/insights/ и делает коммит в main. Сайт обновляется автоматически.
