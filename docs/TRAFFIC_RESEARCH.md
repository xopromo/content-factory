# 🚀 Traffic Specialist Research Agent

Агент-исследователь для изучения стратегий таргетированной рекламы в VK, Threads и Яндекс Директ.

## 📋 Что это?

Это полностью автоматизированный агент, который:
- 🔍 Собирает лайфхаки и лучшие практики по рекламе
- 📝 Ведет подробные логи всех исследований
- 💭 Анализирует и размышляет о найденных инсайтах
- 📊 Структурирует знания по платформам и категориям
- 🔄 Обновляется автоматически

## 🎯 Платформы

### 📘 VK (ВКонтакте)
- Таргетирование по интересам и демографии
- Оптимизация бюджета и СРС
- Оптимальное время для запуска кампаний
- А/В тестирование креативов

### 🧵 Threads
- Вирусный потенциал и reach
- Оптимизация хэштегов
- Монетизация контента
- Органический рост аудитории

### 🔍 Яндекс Директ
- Исследование ключевых слов в Wordstat
- Bidding стратегии и управление ставками
- Минус-слова и фильтрация трафика
- Расширения объявлений и оптимизация CTR

## 📊 Структура данных

### Инсайты
Каждый инсайт содержит:
```json
{
  "id": 1,
  "platform": "VK",
  "title": "Название инсайта",
  "content": "Подробное описание...",
  "category": "targeting",
  "confidence": 7,
  "discovered_at": "2026-03-27T23:53:42...",
  "tags": ["audience", "roi"]
}
```

**Категории:**
- `targeting` - таргетирование аудитории
- `budget_optimization` - оптимизация бюджета
- `timing` - оптимальное время публикации
- `content_strategy` - стратегия контента
- `discovery` - обнаруживаемость контента
- `monetization` - монетизация
- `keywords` - исследование ключевых слов
- `bidding` - управление ставками
- `filtering` - фильтрация трафика
- `optimization` - оптимизация

### Логирование
Все действия агента логируются в JSON:
```
docs/research/logs/session_{id}.json
```

Каждый лог содержит:
- История всех действий агента
- Найденные инсайты
- Статистика исследования
- Временные метки

## 🚀 Использование

### Один раз запустить агента
```bash
python3 traffic_researcher.py
```

### Запустить с логированием в файл
```bash
python3 traffic_researcher.py | tee traffic_research.log
```

## 🔄 Автоматизация

Чтобы агент работал постоянно:

### Вариант 1: Cron job (Linux/Mac)
```bash
# Каждый день в 09:00
0 9 * * * cd /home/user/content-factory && python3 traffic_researcher.py

# Каждый час
0 * * * * cd /home/user/content-factory && python3 traffic_researcher.py
```

### Вариант 2: GitHub Actions
Используй `.github/workflows/traffic-research.yml`

### Вариант 3: Service (systemd)
Создай `/etc/systemd/system/traffic-researcher.service`

## 📈 Метрики и статистика

Агент отслеживает:
- Общее количество инсайтов
- Распределение по платформам
- Распределение по категориям
- Уровень уверенности (confidence) 1-10
- Последнее время обновления

## 💡 Как использовать инсайты

1. **Откройте** `docs/traffic-research.html` в браузере
2. **Изучите** инсайты по платформам
3. **Выберите** релевантные для вашей ниши
4. **Тестируйте** на своих кампаниях
5. **Документируйте** результаты

## 🎓 Категория "Confidence"

Уверенность (1-10) показывает, насколько проверена информация:
- **7-8** - Хорошо проверенные лайфхаки
- **8-9** - Экспертные советы
- **9-10** - Результаты из реальных кейсов
