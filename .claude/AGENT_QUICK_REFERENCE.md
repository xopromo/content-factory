# 🎯 ШПАРГАЛКА: Что запомнить о агентах

## 📝 ОДНА КОМАНДА ДЛЯ ВСЕГО

```bash
./new-agent
```

Ответь на 3 вопроса → агент готов! ✅

---

## 🚀 ВСЕ КОМАНДЫ (на одной странице)

| Что сделать | Команда | Примечание |
|---|---|---|
| **Создать агента** | `./new-agent` | Интерактивный диалог |
| **Создать быстро** | `bash create-agent.sh "имя" "описание"` | Без вопросов |
| **Запустить pipeline** | `bash run-agent-pipeline.sh "вопрос" budget` | Полный цикл: research → judge |
| **Запустить агента** | `@"имя" запрос` | В диалоге |
| **Отредактировать** | Открой `.claude/agents/[name]/[name].md` | В редакторе |
| **Добавить фичу** | Отредактируй `.claude/agents/base-config.yml` | Все агенты получат |
