# 📚 Примеры использования агента фрилансер

## Пример 1: Нужны лайфхаки для увеличения CTR

**Запрос:**
```
@"фрилансер" какие есть лайфхаки для увеличения CTR в таргете ВК? Ищи реальные примеры и инсайты
```

**Ожидаемый результат:**
```
📱 Инсайты про таргет ВКонтакте

💡 Лайфхаки и трюки:
1. Сегментация аудитории по интересам
   → Увеличивает CTR на 35-50%
   → Пример: один клиент за неделю поднял с 2% до 3.5%

2. A/B тестирование креативов
   → Экономит бюджет, находит лучший вариант
   → Результат: поднять конверсию на 25-30%
```

## Пример 2: Нужен кейс для конкретной ниши

**Запрос:**
```
@"фрилансер" найди крутые кейсы про таргет ВК в нише e-commerce (интернет магазины). Что они делали правильно?
```

**Ожидаемый результат:**
```
📊 Реальные кейсы:

1. **Fashion бренд X увеличил продажи на 250%**
   - Ниша: одежда и аксессуары
   - Задача: низкий ROI, много траты без результатов
   - Решение: пересегментировали аудиторию, A/B тестировали,
     оптимизировали по покупкам (не по просмотрам)
   - Результат: +250% продаж за 3 месяца, ROI поднялся с 1:2 на 1:5
```
