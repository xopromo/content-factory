# 🎮 Фрилансер: Автоматический поиск инсайтов ВК Targeting

## 📖 Обзор

Это не просто агент - это **автономный герой**, который сам исследует мир таргетированной рекламы ВКонтакте.

**Главная миссия:** Обогащаться уникальными знаниями об эффективности рекламы

**Как работает:**
1. 🎯 Сам выбирает квесты (что исследовать)
2. 🔍 Ищет информацию в интернете
3. 📝 Логирует каждую находку
4. 👍 Учится на основе твоих оценок (лайки/дизлайки)
5. 🧠 Адаптирует стратегию поиска

## 📋 Система квестов

### Как работает

Каждый квест = задача для исследования

```json
{
  "id": "quest_001",
  "title": "Найди лайфхаки для увеличения CTR",
  "status": "in_progress",
  "priority": 5,
  "evidence_count": 0,
  "rating": 0
}
```

**Статусы:**
- `in_progress` - агент ищет
- `completed` - закончен
- `pending` - ждёт своей очереди

**Приоритет (1-5):**
- 5 = Срочно (темы которые ты лайкал)
- 3 = Стандартный
- 1 = Низкий (темы которые ты дизлайкал)

### Агент САМИ создаёт новые квесты

Когда агент находит что-то интересное, он анализирует:
- "Это новая тема?"
- "Есть ли уже квест на эту тему?"
- Если нет → **сам создаёт квест!**

**Пример:**
```
Найдено: "Эффект видео контента в таргете ВК"
  ↓
Агент думает: "Это новая тема? Есть ли уже квест?"
  ↓
Нет квеста → создаёт: "Собери кейсы про видео контент в ВК"
```

## 👍 Система рейтингов (Лайки и Дизлайки)

### На Dashboard

Когда видишь находку:

```
"Лайфхак: Оптимизация по ЧВП увеличивает ROAS на 35%"

[👍] [👎]  ← Нажми на то, что нравится/не нравится
```

### Что происходит после

**Если ты лайкаешь (👍):**
```
rating: +1
  ↓
Агент видит это и:
  - Повышает приоритет похожих квестов
  - Ищет больше материалов на эту тему
  - Добавляет тему в learned_patterns.json
```

**Если ты дизлайкаешь (👎):**
```
rating: -1
  ↓
Агент видит это и:
  - Снижает приоритет похожих квестов
  - Будет меньше искать такого
  - Помечает тему как "игнорировать"
```

## 📊 Логирование

### Что логируется

```json
{
  "log_id": "log_001",
  "timestamp": "2026-03-26T10:30:00Z",
  "quest_id": "quest_001",
  "finding": "Лайфхак: Оптимизация по ЧВП увеличивает ROAS на 35%",
  "summary": "Вместо оптимизации по кликам выбери оптимизацию по ЧВП",
  "source_url": "https://example.com",
  "content": "[полный текст находки]",
  "rating": 0,
  "user_feedback": null,
  "relevance_score": 0.98,
  "category": "лайфхак"
}
```

### Где находятся логи

- **logs.json** - все логи с рейтингами
- **quests.json** - список квестов
- **learned_patterns.json** - что агент выучил о твоих предпочтениях

## 🔄 ЦИКЛ РАБОТЫ

### Каждые 5 минут (для тестов):

```
1. ИНИЦИАЛИЗАЦИЯ (30 сек)
   └─ Выбери квест с наивысшим приоритетом
   └─ Проверь что нравится пользователю

2. ПОИСК (3-4 минуты)
   └─ WebSearch/WebFetch по теме квеста
   └─ Найди 2-5 релевантных материалов

3. ЛОГИРОВАНИЕ (1 минута)
   └─ Запиши каждую находку в logs.json
   └─ Проверь появился ли новый паттерн
   └─ Может нужен новый квест?

4. АДАПТАЦИЯ (30 сек)
   └─ Прочитай твои лайки/дизлайки
   └─ Обнови приоритеты квестов
   └─ Обнови learned_patterns.json

5. ПЕРЕХОД К СЛЕДУЮЩЕМУ КВЕСТУ
   └─ Повтори цикл
```

## 🧬 LEARNED PATTERNS

Агент помнит что тебе нравится:

```json
{
  "liked_topics": {
    "сегментация": 0.95,
    "A/B тестирование": 0.90,
    "ретаргетинг": 0.85
  },
  "disliked_topics": {
    "спам методы": -0.90,
    "чудо способы": -0.85
  },
  "preferred_content_types": {
    "кейсы": 0.9,
    "лайфхаки": 0.85,
    "шутки": 0.5
  }
}
```

На основе этого агент:
- Повышает приоритет понравившихся тем
- Снижает приоритет не понравившихся
- Ищет больше похожего контента
- Адаптирует стратегию
