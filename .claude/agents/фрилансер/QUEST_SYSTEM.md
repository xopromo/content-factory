## 🤖 Как агент САМИ генерирует задачи

### Шаг 1: Анализ найденного
Когда агент находит интересное, он:
1. Логирует находку в `logs.json`
2. Проверяет: "А есть ли уже квест на эту тему?"
3. Если нет → **сам создаёт новый квест!**

### Шаг 2: Примеры автогенерируемых квестов

**Нашел:** "Эффект сегментации аудитории"
→ Генерирует квест: "Найди все способы сегментации в таргете ВК"

**Нашел:** "Проблема с pixel tracking"
→ Генерирует квест: "Собери кейсы про tracking и attribution"

**Нашел:** "Тренд на видео контент"
→ Генерирует квест: "Какие видео форматы работают лучше всего"

### Шаг 3: Приоритизация
```
Priority = базовый приоритет + рейтинг от пользователя

Высокий приоритет (5):
- Темы которые ты лайкал
- Связано с главной целью (эффективность рекламы)
- Много доказательств уже найдено

Средний приоритет (3):
- Стандартные поиски
- Новые темы

Низкий приоритет (1):
- Темы которые ты дизлайкал
- Мало релевантны к цели
```

## 👍 СИСТЕМА РЕЙТИНГА (Лайки и Дизлайки)

### Как это работает

```
Ты видишь находку на Dashboard:
  "Кейс: Увеличение ROAS на 400% через ретаргетинг"

Ты нажимаешь: 👍 (лайк)
  ↓
Система обновляет:
  - rating: +1 для этой находки
  - Увеличивает приоритет похожих квестов на 2 уровня
  - Добавляет в learned_patterns.json:
    "ретаргетинг" → вес +0.5
```

### Примеры влияния

**ТЫ ЛАЙКАЕШЬ:**
```
👍 "Лайфхак: A/B тестирование креативов"
  ↓
Агент в следующем цикле:
  - Повысит приоритет квестов про A/B тестинг
  - Будет искать больше материалов о тестировании
  - Добавит в список: "A/B тестирование креативов" как ключевая тема
```

**ТЫ ДИЗЛАЙКАЕШЬ:**
```
👎 "Спам о чудо-способах заработка"
  ↓
Агент в следующем цикле:
  - Снизит приоритет похожих тем до 1
  - Исключит такие источники из поиска
  - Отметит в learned_patterns: "спам контент" → игнорировать
```

## 🔄 ЦИКЛ РАБОТЫ АГЕНТА

### Каждые 5 минут (для тестов):

```
1. ИНИЦИАЛИЗАЦИЯ (30 сек)
   └─ Прочитай quest list
   └─ Выбери квест с наивысшим приоритетом
   └─ Проверь learned_patterns (что нравится пользователю)

2. ПОИСК (3-4 минуты)
   └─ Выполни WebSearch/WebFetch по теме квеста
   └─ Найди 2-5 материалов
   └─ Оцени релевантность

3. ЛОГИРОВАНИЕ (1 минута)
   └─ Запиши каждый найденный материал в logs.json:
      {
        "log_id": "log_2026_03_26_10_30",
        "timestamp": "2026-03-26T10:30:00Z",
        "quest_id": "quest_001",
        "finding": "Лайфхак: Оптимизация бюджета по ЧВП",
        "source_url": "https://...",
        "summary": "Компактное описание (2-3 предложения)",
        "content": "Полный контент находки",
        "rating": 0,
        "user_feedback": null,
        "relevance_score": 0.95
      }
   └─ Обнови quest count для этого квеста
   └─ Проверь: появился ли новый pattern?
      └─ Да → создай новый квест

4. АДАПТАЦИЯ (30 сек)
   └─ Прочитай твои лайки/дизлайки из Dashboard
   └─ Обнови приоритеты квестов
   └─ Обнови learned_patterns.json
   └─ Сохрани всё в git

5. ПЕРЕХОД К СЛЕДУЮЩЕМУ КВЕСТУ
   └─ Перейди к следующему по приоритету
   └─ Или заверши текущий и создай новый
```

## 🎮 ИНТЕРПРЕТАЦИЯ КАК РОЛЕВОЙ ИГРЫ

**Агент = Герой 🗡️**
- Имеет главную задачу (искать знания об эффективности рекламы)
- Сам выбирает квесты (что исследовать)
- Учится на основе твоих реакций (лайки/дизлайки)
- Адаптирует стратегию поиска

**Ты = Гайдмастер 🧙**
- Смотришь как герой исследует
- Даёшь обратную связь (👍 или 👎)
- Герой учится у тебя и становится лучше

**Dashboard = Журнал путешествий 📖**
- История всех приключений (логи)
- Карта исследованного (квесты)
- Рейтинги находок
- Статистика героя
