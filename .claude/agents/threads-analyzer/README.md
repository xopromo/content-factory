# 📱 Threads Analyzer Agent - Руководство

**Автоматический мониторинг трендов в Threads для маркетологов!**

## Быстрый старт

### Одноразовый анализ
```
@"threads-analyzer" найди топ-10 виральных постов маркетологов в Threads
```

### Ежедневный мониторинг
```
/loop 1d @"threads-analyzer" найди новые виральные посты маркетологов и предложи insights
```

### Почасовой мониторинг трендов
```
/loop 1h @"threads-analyzer" мониторь Threads для новых виральных постов
```

## Что агент находит

**Категории контента:**
- 📱 SMM и социальные сети
- 🎯 Таргетирование и реклама
- ✍️ Копирайтинг и контент
- 💰 Продажи и продажи
- 🎨 Брендинг и дизайн
- 📊 Аналитика и метрики

**Метрики популярности:**
- 💬 Количество комментариев
- ❤️ Лайки
- 🔄 Репосты
- 📈 Скорость роста (новые посты взлетают быстро?)

## FAQ

**Q: Как агент находит посты если нет Threads API?**
A: Использует WebSearch для поиска информации о популярных постах (гугл индексирует Threads контент).

**Q: Насколько акцуратна информация?**
A: Хорошо для трендов и паттернов, но метрики (точное кол-во лайков) примерные.

**Q: Может ли постить автоматически?**
A: Нет, только предлагает идеи. Ты сам решаешь что постить через @"threads-poster".

**Q: Работает ли ночью?**
A: Да, если запустить `/loop 1d` - будет ежедневно в одно время.

**Q: Где сохраняются результаты?**
A: В JSON файлах в папке агента и MEMORY.md для истории.
