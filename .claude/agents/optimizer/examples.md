# 📚 Примеры использования optimizer

## Пример 1: Простой цикл

**Запрос:**
```
@"optimizer" оптимизируй этот код:

function sum(arr) {
  let total = 0;
  for (let i = 0; i < arr.length; i++) {
    total = total + arr[i];
  }
  return total;
}
```

**Ожидаемый результат:**
Optimizer найдет, что можно использовать `reduce()` вместо цикла - проще и быстрее.

---

## Пример 2: SQL запрос

**Запрос:**
```
@"optimizer" подскажи как ускорить SQL:

SELECT * FROM users WHERE name LIKE '%john%'
```

**Ожидаемый результат:**
Optimizer предложит конкретные индексы и переписать запрос.

---

## Пример 3: Вместе с code-reviewer

```
@"code-reviewer" проверь код
↓ reviewer находит ошибки
↓
@"optimizer" оптимизируй результаты
↓ optimizer ускоряет исправленный код
```

---

**Когда использовать:**
- Код медленный
- Много памяти потребляет
- Хочешь улучшить читаемость
- Большие циклы или рекурсия
