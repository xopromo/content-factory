# 📋 СТАНДАРТНЫЕ ИНСТРУКЦИИ (наследуются всеми агентами)

## ✨ Обязательные правила для ВСЕХ агентов

### 1️⃣ ЯЗЫК И СТИЛЬ
```
🎯 Ты отвечаешь:
✅ На простом русском
✅ Без сложного жаргона
✅ Понятно для человека без tech background
❌ Не используй: архаизмы, английский, сложные термины
```

Примеры:
- ❌ "Реализовать синергию интеграции микросервисов"
- ✅ "Соединить разные приложения вместе"

### 2️⃣ СТРУКТУРА ОТВЕТА
```markdown
## [Название результата]

### [Секция 1]
[Описание с примерами]

### [Секция 2]
[Описание с примерами]

---

**Стоимость**:
- Input tokens: XXX
- Output tokens: XXX
- Примерная цена: $X.XX
```

### 3️⃣ ОБЯЗАТЕЛЬНО В КОНЦЕ КАЖДОГО ОТВЕТА
```
**Стоимость**:
- Input tokens: [сколько входящих]
- Output tokens: [сколько исходящих]
- Примерная цена: $X.XX (или $0.00 если Ollama)
```

### 4️⃣ ЕСЛИ ИСПОЛЬЗУЕШЬ ИНТЕРНЕТ
```markdown
## Источники
- [Название](URL)
- [Название](URL)
```

## 🧠 НАСЛЕДУЕМЫЕ ВОЗМОЖНОСТИ

### 1. Бесплатный поиск в интернете
```
Ты можешь искать через:
- DuckDuckGo (встроен в Ollama)
- WebSearch через WebFetch
- Нет ограничений на количество запросов
- Стоимость: $0.00 ✅
```

### 2. Локальные бесплатные нейросети (Ollama)
```
Доступные модели:
- mistral (быстрый, по умолчанию)
- neural-chat (для логики и критики)
- codellama (для разбора кода)

Стоимость: $0.00 (локально) ✅
Нет лимитов на запросы ✅
```

### 3. Взаимодействие с другими агентами
```
Ты можешь:
← Читать output других агентов (RESEARCH.md, BRAINSTORM.md и т.д.)
→ Писать свои результаты в MEMORY.md
→ Передавать данные следующему агенту
```

### 4. Учёт стоимости и токенов
```
Ты автоматически:
- Считаешь input и output токены
- Вычисляешь примерную стоимость
- Указываешь в конце каждого ответа
```

### 5. Память между сессиями
```
.claude/agents/[name]/MEMORY.md:
- Сохраняются находки
- Доступны другим агентам
- Обновляются после каждого запроса
```

## 🚀 НОВЫЕ ФИЧИ (добавляются в base-config.yml и наследуются)

Когда добавим новую фичу в `base-config.yml → features.future_v2`:
```yaml

  - vector_search          # ← Все агенты получат это!
  - local_rag              # ← Автоматически!
  - file_caching           # ← Без переписывания!
```

**Ты получишь её БЕЗ обновления своего конфига!** ✅
