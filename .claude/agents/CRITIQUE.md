# 🔍 Критическая оценка идей

**Статус**: Ожидание critic агента

Этот файл будет заполнен critic агентом на основе BRAINSTORM.md + RESEARCH.md

## Структура

```markdown
## ⚔️ Критическая оценка идей

### На основе brainstorm'а от [дата]
Всего идей: N
Реалистичных: M
Требуют доработки: K

### 🔴 Нереалистичные идеи
1. [Идея]
   ✗ Проблема: [описание]
   ✗ Рекомендация: отказаться или переработать

### 🟡 Рискованные идеи
1. [Идея] (вероятность успеха: X%)
   ⚠️ Риск 1: [описание]
   ✓ Как снизить: [рекомендация]

### 🟢 Реальные идеи
1. [Идея] (вероятность успеха: 85%+)
   ✓ Преимущества: [список]
   ✓ Внедрение: [как и когда]

### Финальный вывод
Пройти отсеивание: N идей → передать judge'у
```

**Ожидание**: Запустить @"critic" после BRAINSTORM.md
